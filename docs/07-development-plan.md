# 开发计划与排期

## 整体周期

预计 5 周完成一期,按模块逐步交付。

## 阶段划分

### 第一阶段:基础框架(W1 D1-D3)

| 任务 | 产出 | 工时 |
|------|------|------|
| 后端项目初始化 | FastAPI + SQLAlchemy + Alembic 骨架 | 0.5d |
| 数据库建表 | 全量 DDL + 首次迁移 | 0.5d |
| 认证模块 | 登录/注销/JWT/密码哈希 | 1d |
| 前端项目初始化 | Vue3 + Vite + Element Plus + Router + Pinia | 0.5d |
| 前端登录页 | 登录表单 + token 管理 + 路由守卫 | 0.5d |

### 第二阶段:机构与用户(W1 D4 - W2 D2)

| 任务 | 产出 | 工时 |
|------|------|------|
| 机构树 CRUD 后端 | 物化路径维护、新建时复制默认数据 | 1d |
| 机构树前端 | 树形组件、新增/编辑/删除、Logo 上传 | 1d |
| 用户管理后端 | CRUD + 角色分配 + 权限校验中间件 | 1d |
| 用户管理前端 | 列表、新增、编辑、重置密码 | 1d |

### 第三阶段:业务基础数据(W2 D3 - W3 D1)

| 任务 | 产出 | 工时 |
|------|------|------|
| 费用项字典后端 | CRUD + 机构隔离 | 0.5d |
| 费用项字典前端 | 列表、新增、编辑、启用/停用 | 0.5d |
| 公章管理后端 | 上传、CRUD、设置默认 | 0.5d |
| 公章管理前端 | 列表、上传、预览 | 0.5d |
| 系统默认配置后端 | 默认费用项/模板维护接口 | 0.5d |
| 系统默认配置前端 | 管理界面 | 0.5d |

### 第四阶段:模板系统(W3 D2 - W3 D5)

| 任务 | 产出 | 工时 |
|------|------|------|
| XML 解析器(后端) | Python XML → 对象 → HTML 渲染 | 1d |
| XML 解析器(前端) | TypeScript XML → 对象 → HTML 预览 | 1d |
| 模板管理 CRUD | 后端接口 + 导入导出 XML | 0.5d |
| 模板编辑器前端 | 表单式编辑 + 实时预览 | 1.5d |

### 第五阶段:单据填报与导出(W4 D1 - W4 D5)

| 任务 | 产出 | 工时 |
|------|------|------|
| 单据 CRUD 后端 | 创建/查询/修改/删除 + 单号生成 + 修改日志 | 1.5d |
| 填报表单前端 | 费用项下拉、数量加减、金额自动计算 | 1.5d |
| 单据列表/详情前端 | 列表筛选、详情查看、修改 | 1d |
| PDF 导出 | Playwright 渲染 HTML → PDF | 0.5d |
| Excel 导出 | openpyxl 生成 | 0.5d |

### 第六阶段:联调与完善(W5)

| 任务 | 产出 | 工时 |
|------|------|------|
| 前后端联调 | 接口对接、异常处理 | 1d |
| 权限全流程测试 | 管理员/用户场景覆盖 | 1d |
| 打印功能 | 浏览器原生打印 + 打印样式优化 | 0.5d |
| Bug 修复 + 体验优化 | - | 1.5d |
| 部署文档 | Docker Compose 或手动部署指南 | 0.5d |

## 二期规划(一期完成后)

| 模块 | 内容 | 预估 |
|------|------|------|
| Electron 打包 | 单机桌面版 | 1w |
| 模板拖拽编辑器 | 可视化定位 logo/章/字段 | 2w |
| 文件入库 | 文件 JSON 存数据库,前端 Web 渲染 | 1w |
| 数据统计 | 费用汇总、趋势图表 | 1w |

| M3 | W4 结束 | 能填报单据并导出 PDF/Excel |
