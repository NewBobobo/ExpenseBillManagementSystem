# 费用单管理系统 — 设计理念

## 项目名称

费用单管理系统

## 产品定位

面向连锁门店/多级机构的费用单据填报、管理与打印系统。支持多机构层级隔离、权限管控、模板自定义、单据导出。

## 核心设计理念

### 1. 机构隔离,层级穿透

- 每个机构(门店/区域/总部)拥有独立的数据空间:费用字典、模板、公章、单据
- 同级机构之间完全隔离,互不可见
- 管理员可向下穿透管理所有子机构
- 普通用户严格限制在本机构,无论权限如何配置

### 2. 开箱即用,独立运营

- 新建机构时自动从系统默认配置复制一份费用字典和模板
- 各机构独立维护自己的字典和模板,修改不影响其他机构
- 降低初始化成本,同时保留灵活性

### 3. 模板驱动的单据生成

- 单据样式由 XML 模板定义,前端解析渲染预览,后端解析生成 PDF
- 模板可导入导出(XML 文件),便于跨机构复用
- 表单式编辑器降低模板维护门槛

### 4. 双形态部署

- Web 部署:后端服务 + 前端静态资源,多终端浏览器访问
- 单机部署:Electron 打包,内嵌后端服务,双击即用
- 同一份代码,通过构建配置切换部署形态

### 5. 渐进式存储

- 一期:本地磁盘存储文件(logo、公章、PDF)
- 二期:文件以 JSON/二进制存入数据库,前端 Web 渲染,支持服务器集中部署

## 角色模型

| 角色 | 定义 | 能力边界 |
|------|------|----------|
| 管理员 | 机构管理者 | 管理本机构及所有下级:机构信息、用户、权限、字典、模板 |
| 用户 | 业务操作者 | 仅限本机构:填报、查询、修改单据、维护字典、设计模板 |
